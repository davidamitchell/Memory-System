# Projects

Store project-specific context, decisions, and specs here as [memory files](../glossary/memory-file.md).

## File naming convention

```
YYYY-MM-DD-<kebab-case-title>.md
```

## Example front matter

```yaml
---
title: "Open-Brain initial architecture decision"
date: 2025-06-15
tags: [project, architecture, open-brain]
superseded_by: ""
---
```
