# Knowledge

**Domain:** Knowledge representation and engineering

---

## Conceptual Definition

Knowledge is the set of verified, true statements about a domain, understood in their relational context, that enable an agent to reason, infer, and act correctly within that domain.

**Necessary conditions:**
1. **Verified** — each statement has been tested or confirmed against evidence or a trusted source; it is not merely asserted.
2. **True** — the statements accurately correspond to the state of the domain.
3. **Relational** — statements are understood in terms of their connections to other statements and concepts; isolated facts without relational context do not constitute knowledge.
4. **Enabling** — it supports reasoning, inference, or action that could not be performed without it.
5. **Domain-bounded** — its scope and validity are defined relative to a domain.

**Sufficient conditions:**  
Any set of verified, true statements about a domain, understood relationally and enabling reasoning or action within that domain, constitutes knowledge.

---

## Operational Definition

Knowledge is operationally identified when:

1. Statements can be queried and new true statements can be derived by inference.
2. The provenance of each statement is traceable (source, verification method).
3. Inconsistency detection is possible: contradictory statements can be identified.
4. An agent using the knowledge base demonstrably performs better in the domain than an agent without it.

---

## Distinguished From

| Term | Distinction |
|------|-------------|
| **Information** | Information is contextualised data that reduces uncertainty; knowledge additionally requires verification, truth, and relational understanding. |
| **Data** | Data is uncontextualised raw material; knowledge is the fully contextualised, verified, relational output. |
| **Fact** | A fact is a single verified true statement; knowledge is the structured, interconnected set of facts and their relational context. |
| **Wisdom** | Wisdom is the evaluative capacity to judge when and how to apply knowledge in complex situations; knowledge is the content that wisdom draws on. |
