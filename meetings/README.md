# Meetings

Store notes from meetings, calls, and conversations here as [memory files](../glossary/memory-file.md).

## File naming convention

```
YYYY-MM-DD-<kebab-case-title>.md
```

## Example front matter

```yaml
---
title: "Weekly sync with product team"
date: 2025-06-15
tags: [meeting, product]
superseded_by: ""
---
```
